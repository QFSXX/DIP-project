## 2023年冬季学期数字图像处理课程项目————运动检测与行人跟踪  
### 一、实现方法
对图像进行预处理（直方图均衡化），整合现有BoT-Sort跟踪器算法，微调yolov8m模型，实现目标检测与行人跟踪。
### 二、实现功能
#### 1.检测图像中行人，汽车和骑车人，并对其进行轨迹跟踪
#### 2.统计当前帧各类物体数量以及总数
#### 3.检测图像中运动物体（帧差法，不完善）
